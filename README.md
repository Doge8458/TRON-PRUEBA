# TRON-PRUEBA
prueba de un juego html que imita al clasico TRON de arcade

Description

Se añadieron tres archivos nuevos: index.html con la UI y el canvas, styles.css con la estética oscura/neón y diseño responsive, y script.js con la lógica del juego.
La lógica implementa una rejilla (grid) basada en cellSize, movimiento continuo de dos jugadores, marcado de trazas, y comprobación de colisiones contra paredes y trazas.
Controles incluidos: Jugador 1 W A S D y Jugador 2 Flechas, validación que evita reversa instantánea y manejo de estado running/alive.
Se añadió un botón de reinicio y mensajes de resultado (¡Jugador 1 gana!, ¡Jugador 2 gana!, Empate por choque.) y efectos visuales de glow para las motos.
